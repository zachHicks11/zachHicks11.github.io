    Notes on commands

    This where the notes will be, Doc may or may not check this. Not sure.

  • body - This where most of your html code is going to be
  • p - a paragraph tag, commonly used to give out information on a website
  • h1-h6 - these will be header tags, h1 being the largest and h6 being the smallest. h6 is actually smaller than the natural paragraph font
  • footer - this tag will always be at the bottom, it can be used for company copy rights or publishers
  • table - a table tag is what is used to create a table, of course, but with that table comes with many following elements
  • caption - the caption tag will create a spot for text, rather at the top of the table to describe what the table is about. a caption can also be for of course detailing other things like lists.
  • tbodytbody - is used the same as the body tag, but just in the table (hint the t before the word 'body')
